I learned several cool things while doing my Week 2 work.

  1. VS code will automatically put the closer for a tag when you type the tag.
  2. This website is great for study music that my brain doesn't get distracted by.
  3. When creating the Table of Contents I made, I learned about how to link to things in other parts of the directory tree.
    • If the page is in the same directory as the page where the link is placed, the link justs needs to be the filename:
      pageinthesamedirectory.html
    • If the page is in a directory within the one the page where the link is placed is, the link must include the directory name with no slash before it, a forward slash after, and the filename:
      "DirectoryinSameDirectoryAsPage/PageInThatDirectory.html"
    • If the page is in a directory further up, but they share a common grandparent directory, use "../" before the folder and then the file name:
      "../DirectoryInSame ParentDirectoryAsPageLinkedFrom/PageInTgatOtherDirectory.html"
  4. If you click a link on your local page when testing it (I did this on Chrome), and it won't load when you click on it, try opening it in a new tab. There's security stuff in Chrome causing this.
